# 进制

## 常见的进制有二进制、八进制、十进制、十六进制
十进制逢十进一、十六进制逢十六进一

## Python3中进制的转换
- `0b`表示二进制，0b10 --- 输出结果为2
- `0o`表示八进制，0b10 ---输出结果为8
- `0x`表示十六进制，0x10 --- 输出结果为16

## 进制转换
1. 1002转换为十进制

```
  1001(0b1001)
= 2*2^0 + 0*2^1 + 0*2^2 + 1*2^3
= 2 + 0 + 0 + 8
= 10
```

2. 0x 7f(0111 1111)转为十进制

```
  0111 1111
= 7*16^1 + (8+4+2+1)*16^0
= 112+15
= 127
```
3. 127转为二进制

```
  127/2
= 63  % 1
= 31  % 1
= 15  % 1
= 7   % 1
= 3   % 1
= 1   % 1
=     % 1
= 1111111

```
